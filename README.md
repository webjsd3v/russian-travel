# Проект: «Путешествие по России»

Цель проекта «Путешествие по России» показать самые красивые туристические места родной страны. 

## Технологии, которые использовались

* Флексбокс верстка
* Адаптивная верстка
* Гриды
* БЭМ
* Кейфреймы
* Медиазапросы
